myFirstPost

第一次使用hexo进行博客编写并进行测试

使用过程

  • 1 安装nodejs环境后，查看node版本
    • node -v
  • 安装cnpm
    • npm install cnpm -g –registry=https://registry.npm.taobao.org
  • 安装 Hexo
    • cnpm i -g hexo-cli
    • 查看版本：hexo -v
  • 创建本地博客
    • cd d:/hexo/blog
    • hexo init
    • cnpm i
    • hexo s // 打开环境
  • 创建新的文章
    • cd d:/hexo/blog
    • hexo new “My First Post”
  • 创建静态文件
    • hexo g
  • 打开博客
    • 浏览器输入localhost:4000 新文章已出现

部署过程

  • 打开_config.yml文件
    • #Site下
    • title: 博客名
    • subtitle: ‘’
    • description: ‘’ // 描述信息
    • keywords:
    • author: 你的姓名
    • language: zh-Hans // 默认是en
    • timezone: ‘’
    • #URL下
    • url: 网站名称
    • root： 仓库名
    • #Deployer下
    • type: git
    • repository: 仓库链接
    • branch：分支名称
  • 安装自动部署发布工具
    • cnpm i hexo-deployer-git -S
  • 发布
    • hexo clean && hexo g && hexo d
      • 首次发布需要输入账号和密码
  • 打开服务
    • 进入gitee blog仓库的页面->服务->Gitee Pages->启动
